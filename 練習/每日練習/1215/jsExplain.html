前言

JavaScript是一個在WEB上執行的程式語言，與HTML/CSS不同的是，它是一個真正的程式語言，因此你如果開始學習JS的話那麼恭喜你，你現在已經成為一位真正的網頁製作人員。

網站設立的初衷是與使用者互動為目標，JS在網頁中扮演協助使用者與網站雙方互動的角色，沒有JS的網站稱為靜態網站。

在開始撰寫JavaScript前，建議先安裝vs code套件JavaScript (ES6) 在使用時會提示語法幫助撰寫速度，但仍然強烈建議須將完整名字了解記住。

JavaScript使用說明

JS的使用方式分為script標籤內以及script src導入的方式將JS檔導入置網頁，早期網站會將src位置放在head標籤內但隨著網頁技術進步，需要的JS檔案越來越多導致載入的時間被拉長因此src位置放在/body標籤前可以優先載入文字與畫面再載入功能，在開始前必須先了解瀏覽器中的開發人員工具console。

在JS中使用console.log(對象)後，console面板會顯示對象內的值或執行運算結果，是一個非常實用的功能，在實務上可以透過此方式得知變數當下的值或是有哪些參數可以使用。

變數宣告

  1. var

    2. 作用域為全域可以被重複宣告，在錯誤的觀念使用上容易造成全域變數汙染，支援ES6以上的網站強烈建議改用let

  2. let

    3. 作用域為區域在同一個作用域內不能被重複宣告，宣告的變數只能在作用域內產生作用

  3. const

    4. 又稱常數與let類似，宣告時必須賦予型態或值，且無法透過直接修改的方式對值進行修改

作用域

在最外層的區域稱為全域，使用{}包起來的區域稱為區域，若是在區域內使用不存在的變數則會從上面外層尋找，在進行專案中若非必要盡量在區域變數內宣告及使用變數避免全域汙染導致變數計算結果與期望值不同，試著將以下的let改成var後再console面板看看結果!

  1.         <script>
            let a = 1  //我是全域變數
            {  //區域開始
                let a = 2  //我是區域變數
                console.log('內',a)
            }  //區域結束
            console.log('外',a)
        </script>

    2. 透過上述範例可得知let在不同的作用域不會互相干擾，而var則會，對於入門的初學者或許可能有些難以理解但是作用域引起的BUG會隨著專案的增大而更容易出現，對於作用域的認知一定要理解原理。

陣列與物件

JS中有兩個非常重要的型態就是陣列與物件，陣列用來儲存多筆資料陣列的位置稱為索引值索引值的起始值是0而不是1，物件是一種使用自訂名稱紀錄內容或function函式的資料型態，兩者可以合併使用，試著將以下範例的console做更改。

  1.         <script>
            let group = ['小美','小明','瑪琳','阿力']
            console.log(group)
            console.log(group[2])
        </script>

    2. 不使用索引值印出陣列是印出整個物件內容，而使用索引值則會印出對應的陣列位置內容，最常見使用方法如迴圈等方式對陣列內全部內容進行事件處裡。

  2.         <script>
            let groupObj = [
                {
                    name:'小美',
                    age:20,
                    gender:'女'
                },
                {
                    name:'小明',
                    age:23,
                    gender:'男'
                },
                {
                    name:'瑪琳',
                    age:18,
                    gender:'女'
                },
                {
                    name:'阿力',
                    age:30,
                    gender:'男'
                }
            ]
            console.log(groupObj[0])
            console.log(groupObj[0].name)
        </script>

    3. 物件中需用.名稱的方式呼叫儲存的對應資料，在這邊是常見利用陣列與物件的特性將資料做整合紀錄，再使用迴圈功能針對資料做處理如渲染至畫面上或批量修改等。
